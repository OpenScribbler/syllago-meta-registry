# syllago-meta-registry

Official syllago meta-registry — skills, agents, loadouts, and content for syllago usage and management

## Using this registry

```sh
syllago registry add <git-url>
syllago registry sync
```

## Structure

- `skills/` -- Skills
- `agents/` -- Agents
- `mcp/` -- MCP Servers
- `rules/` -- Rules
- `hooks/` -- Hooks
- `commands/` -- Commands
- `loadouts/` -- Loadouts
